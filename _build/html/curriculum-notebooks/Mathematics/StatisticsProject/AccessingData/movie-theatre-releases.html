Movie Theatre Release Numbers¶

We can get data about theatrical release gross profits from Box Office Mojo by IMDBPro.

year = 2019
url = 'https://www.boxofficemojo.com/year/world/'+str(year)
import pandas as pd
box_office = pd.read_html(url)[0]
